# 功能说明

* 管理员/开发者扫二维码登录

* 公众号菜单管理

* 关注人数/用户管理

* 推送管理

* 回复管理/客服沟通

* H5页面管理

# 框架说明

## viewer

* 后台管理系统

* 推送内容编写

* 公众号预览

* H5页面预览

* 通用模板

## controller

* 用户关联

* 公众号页面模拟

* H5页面生成

* 菜单生成/刷新

* 关注用户信息管理

* 用户信息管理

* 推送内容转译

* 回复信息发送/接收

## service

* 微信公众号服务

* 虚拟DOM服务

* socket/http数据传输服务

## database

* userInfo -- 用户登录信息

* roleInfo -- 用户权限信息

* wechatCommonInfo -- 公众号绑定信息

* menuInfo -- 菜单信息

* clientInfo -- 关注人/用户信息

* pushInfo -- 推送信息

* serviceChatInfo -- 对话/客服信息

* modalPageInfo -- 模板页面信息

* pageData -- H5/自定义页面信息
